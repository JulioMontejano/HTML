Se usa la palabra reservada para el color rojo en html.

Se usa el código hexadecimal para dar el color.

Se usa la mezcla de colores RGB (Red, Green, blue) en la cual los valoras van del 0 - 255.

Se usa una mezcla de colores en HSL (Hue, Saturation, lightness) Son valores de cualquier tipo de color y se usan en porcentaje.

Se usa la opacidad como parámetro del color rojo que se eligió de fondo.

En este se usa una variante del anterior RGB pero usando un canal “Alpha” que sirve para crear transparencias.

Y por último se usa el atributo transparent, para quitar el color rojo elegido.
